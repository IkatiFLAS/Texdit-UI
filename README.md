# Textdit-UI

Texdit-UI es un editor de texto ligero inspirado
en la filosofia Unix y en editores como VIM,
HELIX, KAKOUNE.

El objetivo del proyecto es crear un editor rapido,simple
y extensible,construido desde cero en C++ moderno.

## Caracteristicas Actuales
1. Apertura de archivos
2. Sistema de Buffer
3. Sistema de cursor
4. Renderizado en terminal
5. Entrada en modo RAW
6. Navegacion con teclas estilo VIM
7. Sistema de modos (Normal,Insert,Visual)

## Tecnologias

1. C++23
2. CMake
3. Terminal ANSI

## Filosofia

1. Simplicidad
2. Velocidad
3. Legibilidad del codigo
4. Filosofia Unix
5. Dependencias minimas
